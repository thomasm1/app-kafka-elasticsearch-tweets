#### APP:
Cryptomaven [REST-API] Cryptocurrency wallet and tracker
#### MICROSERVICE:  
Persistence API 
* modify/save cryptocurrency data to AWS-hosted MySQL db. 
* modify/save user data to AWS-hosted Mongo/Document dynamo db.
